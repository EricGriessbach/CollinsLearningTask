RLWM Task

Reinforcement Learning & Working Memory Experiment

Welcome to the RLWM Task!

In this experiment, you'll learn associations between symbols and actions through trial and feedback.

  • Goal: Learn which action (1, 2, or 3) is correct for each symbol
  • How to play: When you see a symbol, press 1, 2, or 3 on your keyboard (or click the buttons)
  • Feedback: You'll receive immediate feedback - "Correct!" or "Incorrect"
  • Blocks: You'll complete multiple blocks with different numbers of symbols (2-6 per block)
  • Learning: Within each block, associations stay the same - use feedback to learn!

Note: This is a simplified version of the research task used in:

Collins, A.G.E. (2025). A habit and working memory model as an alternative account of human reward-based learning. Nature Human Behaviour. https://doi.org/10.1038/s41562-025-02340-0

About this task: This is an interactive recreation of the RLWM (Reinforcement Learning & Working Memory) task, designed to study how humans learn from rewards under different memory loads. The original research demonstrated that human reward-based learning relies on a combination of fast working-memory-based processes and slower habit-like associative processes.
